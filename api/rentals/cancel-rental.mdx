---
title: "Cancel Rental"
api: "POST /api/rentals/{id}/cancel"
---

Cancels an existing rental by updating its status to `CANCELLED`.

## Path Parameters

<ParamField path="id" type="long" required>
  The unique identifier of the rental to cancel
</ParamField>

## Response

Returns `204 No Content` on successful cancellation.

<RequestExample>
```bash cURL
curl -X POST https://api.drivex.com/api/rentals/789/cancel \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```bash 204 No Content
```

```json 404 Not Found
{
  "error": "Rental not found"
}
```
</ResponseExample>
