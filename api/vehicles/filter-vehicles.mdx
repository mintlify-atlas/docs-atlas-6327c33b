---
title: "Filter Vehicles"
api: "GET /api/vehicles/filter"
description: "Filter vehicles by multiple criteria"
---

Filters vehicles using multiple optional criteria. All parameters are optional and can be combined. Empty strings are treated as null.

## Query Parameters

<ParamField query="brand" type="string" optional>
  Filter by brand name
</ParamField>

<ParamField query="model" type="string" optional>
  Filter by model name
</ParamField>

<ParamField query="year" type="integer" optional>
  Filter by manufacturing year
</ParamField>

<ParamField query="vehicleType" type="string" optional>
  Filter by vehicle type (e.g., sedan, suv, truck, motorcycle)
</ParamField>

## Response

Returns an array of Vehicle objects matching all specified criteria.

<ResponseField name="id" type="long">
  Unique identifier for the vehicle
</ResponseField>

<ResponseField name="reference" type="string">
  Auto-generated unique reference code
</ResponseField>

<ResponseField name="brand" type="string">
  Vehicle brand name
</ResponseField>

<ResponseField name="model" type="string">
  Vehicle model name
</ResponseField>

<ResponseField name="hp" type="integer">
  Horsepower
</ResponseField>

<ResponseField name="autonomy" type="integer">
  Autonomy/range in kilometers
</ResponseField>

<ResponseField name="averageconsumption" type="double">
  Average fuel consumption
</ResponseField>

<ResponseField name="description" type="string">
  Vehicle description
</ResponseField>

<ResponseField name="price" type="decimal">
  Vehicle price
</ResponseField>

<ResponseField name="year" type="integer">
  Manufacturing year
</ResponseField>

<ResponseField name="fuelType" type="string">
  Fuel type
</ResponseField>

<ResponseField name="mileage" type="long">
  Mileage in kilometers
</ResponseField>

<ResponseField name="extras" type="string">
  Additional features and extras
</ResponseField>

<ResponseField name="createdAt" type="datetime">
  Timestamp when created
</ResponseField>

<ResponseField name="updatedAt" type="datetime">
  Timestamp when last updated
</ResponseField>

<ResponseField name="doors" type="integer">
  Number of doors
</ResponseField>

<ResponseField name="vehicleType" type="string">
  Type of vehicle
</ResponseField>

<ResponseField name="offers" type="string">
  Special offers status
</ResponseField>

<ResponseField name="images" type="array">
  Array of vehicle images
</ResponseField>

<RequestExample>
```bash cURL (single filter)
curl -X GET "http://localhost:8080/api/vehicles/filter?brand=BMW"
```

```bash cURL (multiple filters)
curl -X GET "http://localhost:8080/api/vehicles/filter?brand=BMW&year=2024&vehicleType=SUV"
```

```bash cURL (all filters)
curl -X GET "http://localhost:8080/api/vehicles/filter?brand=BMW&model=X5&year=2024&vehicleType=SUV"
```
</RequestExample>
