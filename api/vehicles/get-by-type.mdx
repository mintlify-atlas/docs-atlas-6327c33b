---
title: "Get Vehicles by Type"
api: "GET /api/vehicles/vehicleType"
description: "Retrieve vehicles by their vehicle type classification"
---

## Request

<ParamField query="q" type="string" required>
  The vehicle type to filter by (e.g., SUV, Sedan, Coupe, Hatchback)
</ParamField>

## Response

Returns an array of vehicles matching the specified type.

<ResponseField name="vehicles" type="Vehicle[]">
  Array of vehicle objects
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8080/api/vehicles/vehicleType?q=SUV"
```

```java Java
// From VehicleController.java:96-99
@GetMapping("/vehicleType")
public List<Vehicle> findByVehicleType(@RequestParam String q) {
    return service.findByVehicleType(q);
}
```

## Common Vehicle Types

- SUV
- Sedan
- Coupe
- Hatchback
- Convertible
- Station Wagon
- Van
- Pickup Truck

<Note>
  Vehicle types are stored as strings in the database. Matching is case-sensitive.
</Note>
