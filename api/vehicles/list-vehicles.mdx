---
title: "List Vehicles"
api: "GET /api/vehicles"
description: "Retrieve a list of all vehicles, optionally filtered by brand"
---

Returns a list of all vehicles in the system, ordered by brand in ascending order. You can optionally filter by brand using a query parameter.

## Query Parameters

<ParamField query="marca" type="string" optional>
  Filter vehicles by brand name (case-insensitive partial match)
</ParamField>

## Response

Returns an array of Vehicle objects.

<ResponseField name="id" type="long">
  Unique identifier for the vehicle
</ResponseField>

<ResponseField name="reference" type="string">
  Auto-generated unique reference code in format: `{BRAND}{YEAR}{RANDOM}` (e.g., BMW2024A1B2)
</ResponseField>

<ResponseField name="brand" type="string">
  Vehicle brand name
</ResponseField>

<ResponseField name="model" type="string">
  Vehicle model name
</ResponseField>

<ResponseField name="hp" type="integer">
  Horsepower
</ResponseField>

<ResponseField name="autonomy" type="integer">
  Autonomy/range in kilometers
</ResponseField>

<ResponseField name="averageconsumption" type="double">
  Average fuel consumption
</ResponseField>

<ResponseField name="description" type="string">
  Vehicle description
</ResponseField>

<ResponseField name="price" type="decimal">
  Vehicle price
</ResponseField>

<ResponseField name="year" type="integer">
  Manufacturing year
</ResponseField>

<ResponseField name="fuelType" type="string">
  Fuel type (e.g., gasoline, diesel, electric, hybrid)
</ResponseField>

<ResponseField name="mileage" type="long">
  Mileage in kilometers
</ResponseField>

<ResponseField name="extras" type="string">
  Additional features and extras
</ResponseField>

<ResponseField name="createdAt" type="datetime">
  Timestamp when the vehicle was created
</ResponseField>

<ResponseField name="updatedAt" type="datetime">
  Timestamp when the vehicle was last updated
</ResponseField>

<ResponseField name="doors" type="integer">
  Number of doors
</ResponseField>

<ResponseField name="vehicleType" type="string">
  Type of vehicle (e.g., sedan, suv, truck, motorcycle)
</ResponseField>

<ResponseField name="offers" type="string">
  Special offers status (e.g., "Yes" or "No")
</ResponseField>

<ResponseField name="images" type="array">
  Array of vehicle images
  <ResponseField name="id" type="long">
    Image ID
  </ResponseField>
  <ResponseField name="imageUrl" type="string">
    Image URL
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "http://localhost:8080/api/vehicles"
```

```bash cURL (filtered by brand)
curl -X GET "http://localhost:8080/api/vehicles?marca=BMW"
```
</RequestExample>
