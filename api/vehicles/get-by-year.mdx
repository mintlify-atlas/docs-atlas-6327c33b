---
title: "Get Vehicles by Year"
api: "GET /api/vehicles/year"
description: "Retrieve all vehicles from a specific manufacturing year"
---

## Request

<ParamField query="year" type="integer" required>
  The manufacturing year to filter by
</ParamField>

## Response

Returns an array of vehicles from the specified year.

<ResponseField name="vehicles" type="Vehicle[]">
  Array of vehicle objects
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8080/api/vehicles/year?year=2024"
```

```java Java
// From VehicleController.java:91-94
@GetMapping("/year")
public List<Vehicle> searchYear(@RequestParam int year) {
    return service.findByYear(year);
}
```
</CodeGroup>
