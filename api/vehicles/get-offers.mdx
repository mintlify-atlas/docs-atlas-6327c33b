---
title: "Get Vehicles with Offers"
api: "GET /api/vehicles/offers"
description: "Retrieve all vehicles that have active promotional offers"
---

## Request

This endpoint requires no parameters.

## Response

Returns an array of vehicles that have offers defined (non-null `offers` field).

<ResponseField name="vehicles" type="Vehicle[]">
  Array of vehicle objects with active offers
</ResponseField>

## Example

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8080/api/vehicles/offers"
```

```java Java
// From VehicleController.java:126-129
@GetMapping("/offers")

    return service.findByOffers();
}
```

<Info>
  This endpoint filters vehicles where the `offers` field is not null. To update offers for a vehicle, use the `PATCH /api/vehicles/{id}/offers` endpoint.
</Info>
