---
title: "Search Vehicles"
api: "GET /api/vehicles/search"
description: "Search vehicles by brand or model name"
---

Searches for vehicles where either the brand or model contains the search term (case-insensitive).

## Query Parameters

<ParamField query="q" type="string" required>
  Search term to match against brand or model (case-insensitive partial match)
</ParamField>

## Response

Returns an array of Vehicle objects matching the search criteria.

<ResponseField name="id" type="long">
  Unique identifier for the vehicle
</ResponseField>

<ResponseField name="reference" type="string">
  Auto-generated unique reference code
</ResponseField>

<ResponseField name="brand" type="string">
  Vehicle brand name
</ResponseField>

<ResponseField name="model" type="string">
  Vehicle model name
</ResponseField>

<ResponseField name="hp" type="integer">
  Horsepower
</ResponseField>

<ResponseField name="autonomy" type="integer">
  Autonomy/range in kilometers
</ResponseField>

<ResponseField name="averageconsumption" type="double">
  Average fuel consumption
</ResponseField>

<ResponseField name="description" type="string">
  Vehicle description
</ResponseField>

<ResponseField name="price" type="decimal">
  Vehicle price
</ResponseField>

<ResponseField name="year" type="integer">
  Manufacturing year
</ResponseField>

<ResponseField name="fuelType" type="string">
  Fuel type
</ResponseField>

<ResponseField name="mileage" type="long">
  Mileage in kilometers
</ResponseField>

<ResponseField name="extras" type="string">
  Additional features and extras
</ResponseField>

<ResponseField name="createdAt" type="datetime">
  Timestamp when created
</ResponseField>

<ResponseField name="updatedAt" type="datetime">
  Timestamp when last updated
</ResponseField>

<ResponseField name="doors" type="integer">
  Number of doors
</ResponseField>

<ResponseField name="vehicleType" type="string">
  Type of vehicle
</ResponseField>

<ResponseField name="offers" type="string">
  Special offers status
</ResponseField>

<ResponseField name="images" type="array">
  Array of vehicle images
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "http://localhost:8080/api/vehicles/search?q=BMW"
```

```bash cURL (search by model)
curl -X GET "http://localhost:8080/api/vehicles/search?q=X5"
```
</RequestExample>
