---
title: "Get User"
api: "GET /api/users/{id}"
description: "Retrieve a specific user by their ID"
---

## Overview

This endpoint retrieves detailed information about a specific user by their unique identifier.

## Request

<ParamField path="id" type="long" required>
  The unique identifier of the user to retrieve
</ParamField>

```bash
curl -X GET 'http://localhost:8080/api/users/1' \
  -H 'Content-Type: application/json'
```

## Response

<ResponseField name="id" type="long">
  Unique identifier for the user
</ResponseField>

<ResponseField name="username" type="string">
  User's username
</ResponseField>

<ResponseField name="email" type="string">
  User's email address
</ResponseField>

<ResponseField name="password_hash" type="string">
  Encrypted password hash
</ResponseField>

<ResponseField name="firstname" type="string">
  User's first name
</ResponseField>

<ResponseField name="lastname" type="string">
  User's last name
</ResponseField>

<ResponseField name="phone_number" type="string">
  User's phone number
</ResponseField>

<ResponseField name="role" type="string">
  User's role in the system (e.g., ADMIN, USER, DRIVER)
</ResponseField>

<ResponseField name="is_active" type="boolean">
  Whether the user account is active
</ResponseField>

<ResponseField name="profileImage" type="string">
  URL or path to the user's profile image
</ResponseField>

<ResponseField name="created_at" type="timestamp">
  Timestamp when the user was created
</ResponseField>

<ResponseField name="updated_at" type="timestamp">
  Timestamp when the user was last updated
</ResponseField>

## Example Response

```json
{
  "id": 1,
  "username": "johndoe",
  "email": "john.doe@example.com",
  "password_hash": "$2a$10$...",
  "firstname": "John",
  "lastname": "Doe",
  "phone_number": "+1234567890",
  "role": "USER",
  "is_active": true,
  "profileImage": "/images/profiles/johndoe.jpg",
  "created_at": "2024-01-15T10:30:00Z",
  "updated_at": "2024-01-20T14:45:00Z"
}
```

## Error Responses

<ResponseField name="404" type="error">
  User not found - returned when no user exists with the specified ID
</ResponseField>
